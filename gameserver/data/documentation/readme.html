Akumu Geodata

Благодарим Вас за приобретение Interlude геодаты от Akumu. Ваш выбор очень важен для нас!

Установка геодаты на вашу сборку

  1. Для начала откройте конфигурационный файл геодвижка вашей сборке (обычно файл находится по адресу ./data/config/geodata.properties), если вам не удаётся найти данный файл, уточните его расположение у разработчиков вашего эмулятора.

  2. Найдите переменную отвечающую за активацию геодвижка вашей сборки. (обычно GeoData)


    # GeoData options: (no recommendations, for each his own)
    # 0 = GeoData and PathFinding OFF (default)
    # 1 = GeoData is used to check Line Of Sight (LOS) targeting and
    # L2Playable movement. You need to download files for data/geodata folder.
    # Monsters can pass walls but not aggro (no line of sight) through them.
    # 2 = Full GeoData enabled. Includes PathFinding (requires also /data/pathnode
    # files if CellPathFinding not enabled) and all character moves go through
    # geodata checks (if a mob passes a wall, pathfinding didn't find a route
    # but we allow attack and returning home).
    # Recommended server memory minimum 2 GB, rather 3 GB.
    GeoData = 0

    Установите значение переменной в 1 для активации использования геодаты, либо 2 для использования геодаты + поиска пути.

  3. Переместите содержание архива геодаты akumu в отдельную папку.
    Скопируйте содержание папок geodata и pathnode в директории ./data/geodata и ./data/pathnode соответственно.


  4. Запустите/перезагрузите свой игровой сервер (GS) для того, чтобы изменения вступили в силу.

  5. В том случае если вы приобрели геодату более недели назад, имеет смысл проверить наличае обновлений на сайте разработчика.

    Последние обновления геодаты: http://l2geo.ru/changelog.html
