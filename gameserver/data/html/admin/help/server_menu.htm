Botones del panel de 4 servidores
Las acciones relacionadas con el servidor L2J están aquí: configuraciones, multiplicadores, caché.
4-1Recargar y otros.
Recargar:Intenta volver a cargar el administrador de instancias especificado. Los valores posibles son: multisell,skill,npc,htm,item,instancemanager,npcwalkers.
Recargar HTM:Limpia el caché HTM.
Recargar archivo:Vaciar la caché del archivo HTM especificado.
Prueba de habilidad:Al igual que //effect, recibe un ID de habilidad como parámetro, afecta al objetivo o al lanzador.
Recargar CW:Recarga el administrador de armas malditas.
Zona de recarga:Recarga el administrador de zonas.
Recargar escudo:Reconstruye la caché de Crest.
Solución a la cresta:Convierte archivos Crest antiguos si se encuentran.
Recargar Tele:Recargar destinos de teletransporte.
Social:Hacer que los personajes reproduzcan animaciones sociales. El cuadro rápido puede contener el identificador social, el nombre del jugador (opcional) y el radio (opcional). Si solo se proporciona un parámetro, afecta al objetivo.
Abrir/Cerrar todo:Abrir o cerrar todas las puertas del servidor.
Comprobación de zona:Realiza comprobaciones de zona a demanda.

Calculadora de lucha:Iniciar diálogos para el cálculo de la pelea entre dos mobs.Forja de paquetes:Inicia las instalaciones de elaboración de paquetes.
Mensaje del sistema:Le envía el mensaje del sistema especificado.
4-4 Mantenimiento.
Estadísticas:Muestra estadísticas del servidor sobre el uso de memoria, tareas, etc.
Acceso:Muestra información y opciones relacionadas con el servidor de inicio de sesión.
Cerrar:Mostrar acciones relacionadas con reiniciar/apagar el servidor.
