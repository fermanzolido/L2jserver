1- Botones del panel principal
Los comandos más útiles se han organizado aquí.
1-1 Las cosas que usarás la mayor parte del tiempo
Artículo:Toma el ID del artículo y, opcionalmente, la cantidad de artículos de QuickBox para crear dicho artículo en tu inventario. Muestra un cuadro de diálogo específico si no se especificó nada.
Teletransportarse:Lee las coordenadas desde QuickBox. Muestra el menú de teletransportación y movimientos si no se especifica ninguno.
Desovar:Toma el ID o el nombre del NPC (los espacios deben cambiarse por guiones bajos); opcionalmente, el número de apariciones y el tiempo de reaparición. Muestra un cuadro de diálogo específico si no hay ninguno.
Lista de genera:Toma el ID o el nombre del NPC (los espacios deben reemplazarse por guiones bajos). Muestra los resultados como mensajes del sistema.
Anunciar:toma el texto para anunciar desde el cuadro rápido; muestra el menú de anuncios si no se especificó ningún texto.
Colocar:Requiere "parámetro=valor" en el cuadro de acceso rápido. Muestra el cuadro de diálogo de configuración si no se especificó nada válido.
Charla del GM:Transmitir contenido de Quickbox solo para GM. El menú de administración aparecerá después de la transmisión.
Lista+Ir:Al igual que el botón Lucky de Google, te teletransportará al primer resultado de un comando List Spawn.
Ir a:Lee el nombre del jugador desde QuickBox. Muestra la pantalla de administración de personajes si no se especifica ninguno.
Recordar:trae un jugador a tu posición, si se especifica un nombre en el cuadro rápido
Karma:Establece puntos de karma especificados a través del cuadro rápido para el objetivo actual (si es un jugador, naturalmente).
Sin karma:Al igual que el botón Karma, funciona sobre el objetivo, pero ignora los valores QB al usar cero.
