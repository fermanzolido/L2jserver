Instrucciones para el control de grupos de mafias.

//crear grupo de mob[ID de grupo] [ID de npc] [número]
Crea un grupo con [número] mobs del tipo [npcId], en el grupo [groupId].
No hay ningún efecto visual al utilizar este comando, y los miembros de este grupo deben generarse para que se active.

//eliminar grupo de mafia[Identificación del grupo]
Elimina/elimina el grupo [groupId], y posteriormente deja de aparecer cualquier miembro del grupo que aún esté vivo.

//grupo de mob_spawn[ID de grupo] [locx] [locy] [locz]
Genera los miembros del grupo [groupId]. Si no se especifica una ubicación, el grupo se generará alrededor del GM.
De forma predeterminada, los mobs están en un estado inactivo y esperan más comandos.
//grupo de mob_desaparecer[Identificación del grupo]
Elimina a los miembros del grupo [groupId]. El grupo no se elimina y se puede invocar mediante el uso de //mobgroup_spawn una vez más.
//lista de grupos de mobs([ID del grupo])
Enumera todos los grupos de monstruos creados. Si se especifica [groupId], se muestra una lista de los miembros de ese grupo.

Otros comandos
//grupo de mob_seguir[groupId] - El grupo [groupId] seguirá al GM.
//grupo de mob_retorno[groupId] - El grupo [groupId] regresará al GM.
//teletransporte de grupo móvil[groupId] ([playerName]) - El grupo [groupId] se teletransportará y seguirá al jugador especificado (o al GM si no se especificó ninguno).
//ataque de grupo de mobs[groupId] - El grupo [groupId] atacará al objetivo actual del GM (si lo hay).
//grupo de mob_rnd[groupId] - El grupo [groupId] atacará objetivos seleccionados al azar.
//grupo de mob_ataque_rp[groupId1] [groupId2] - El grupo [groupId1] atacará a los miembros de [groupId2].
//casting de grupos de mobs[groupId] - Los mobs del grupo [groupId] solo usarán sus habilidades para atacar.
//grupo de mob_nomove[groupId] [on/off] - Activa o desactiva los monstruos del grupo [groupId] para que ya no se muevan. Resulta interesante cuando se utiliza con el modo de solo lanzamiento.
//grupo de mob_invul[groupId] [on/off] - Activa o desactiva los mobs del grupo [groupId] para que sean invulnerables.
//grupo móvil_inactivo[groupId] - Devuelve los mobs del grupo [groupId] a un estado inactivo (en espera).
//asesinato de un grupo de la mafia[groupId] - Mata a todos los mobs vivos en el grupo [groupId].
